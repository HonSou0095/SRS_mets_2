# SRS_mets_2

Code and data for the study:
“Systemic Immune-Inflammation and Response Indices Predict Early but Not Long-Term Survival After Primary Stereotactic Radiosurgery for Brain Metastases.”

This repository contains the Python (Google Colab) notebook used for statistical analyses and figure generation described in the manuscript.

## Contents
- `SRS_mets2_analysis.ipynb`: main analysis notebook

## Requirements
Python 3.12  
Libraries: statsmodels, lifelines, scikit-learn, pandas, seaborn, matplotlib

If you use this code, please cite:
